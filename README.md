### 使用说明

在 `Jenkins` 的【脚本命令行】运行如下命令，查看 `Jenkins` 系统使用的 `Groove` 版本

```
println GroovySystem.version
```

如无特殊情况，共享库使用的 `Groovy` 版本尽量和 `Jenkins` 中的保持一致。
